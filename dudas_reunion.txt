En cada bloque de datos (Light / Dark), en el Branch dataChA hay 10000 muestras. Es correcto que en el
xlabel ponga muestras? Por que en el artículo usan como xlabel unidad de tiempo en microsegundos. (pag. 4)

En la grafica de la media:
 De cada evento / señal, se calcula la media de cada muestra, una por una. Por ejemplo:

 Evento / señal 0 : muestra 0	---\
 Evento / señal 1 : muestra 0	----\
 Evento / señal 2 : muestra 0	----/	Calcula la media de la muestra 0 de los 4722 señales
 Evento / señal 3 : muestra 0 	---/
 Evento / señal 4722 : muestra 0 

Para poner marco temporal en el eje X, basta averiguar cada cuanto tiempo el TES toma una muestra.
